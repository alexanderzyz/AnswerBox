

# 澳猫团购物平台项目



## 实现技术

### 前端

- 根据老师提供的html,css,js,image的基础上进行了修改

- js中主要使用jQuery完成时间绑定，动态添加元素

- 利用ajax和后端进行数据交互，ajax全部采用post方式

### 后端

- JDK11

- 项目管理Maven
- 数据库管理mybatis
- 框架spring+springMVC+springBoot
- 验证码工具easy-captcha和Json工具fastjson
